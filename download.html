Installing Java

Dust is distributed as a JAR file and it uses Java to run. If Java is not already installed on your computed, follow the instructions below:

Window
Download and install Java for Window. Then double click on the .jar file.

Linux
Type sudo apt-get install default-jre in the console. Then start the game with java -jar dust_launcher.jar.
Alternatively, download and install Java for Linux.

Mac OS
Download and install Java for Mac OS.
